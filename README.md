# CivicFix

A community-driven platform empowering citizens to report and resolve civic issues in their neighborhoods.

# Live URL Link: 

## 🚀 Features

- **Report Issues**: Submit civic issues with detailed descriptions, images, and severity levels
- **Browse Issues**: Explore all reported issues in the community
- **Issue Details**: View comprehensive information about each issue including status and priority
- **User Authentication**: Secure login and registration with email/password or Google OAuth
- **Manage Issues**: Track and manage issues you've reported
- **Responsive Design**: Fully responsive UI optimized for all devices


## 🎨 Design & UI

- **Framework**: Next.js with React
- **Styling**: Tailwind CSS + DaisyUI
- **Icons**: React Icons
- **Notifications**: React Hot Toast
